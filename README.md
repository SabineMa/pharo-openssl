This are [Pharo](https://github.com/pharo-project/pharo) bindings for the [openssl]( https://www.openssl.org/)
library, generated using [pharo-cig](https://github.com/estebanlm/pharo-cig).  

The generation recipe can be found at the class comment of [LibSsl class](https://github.com/pharo-cig/pharo-openssl/blob/main/src/LibSsl/LibSsl.class.st).
